BookList by RobvanB

Android app to track your book collection

Android application to track your book collection. Main goal is to quickly look up which books you have/need when in the bookstore. Uses DataCrow (http://www.DataCrow.net) as the main source, and imports data from this application. Uses DropBox (http://www.DropBox.com) to pull the export file from DataCrow into the Android App. Still in development. ^^

Install

Open project in Eclipse and create the APK file. Then install the APK file on your device. Or pull the signed APK file from the repo and install that one.
